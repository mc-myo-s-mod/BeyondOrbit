# Beyond Orbit

Beyond Orbit은 데이터팩으로 정의된 천체에서 자원을 캐는 모드입니다.

처음 보면 블록 이름만 보고 용도를 알기 어렵습니다. 이 가이드는 **무엇을 만들고, 어디에 놓고, 어떤 순서로 쓰는지**를 설명합니다.

## 빠른 길

1. <ItemLink id="beyondorbit:orbital_data_core" />를 만듭니다.
2. <ItemLink id="beyondorbit:basic_satellite" />를 만듭니다.
3. 초반에는 <ItemLink id="beyondorbit:satellite_uplink" />를 설치합니다.
4. Uplink가 에너지를 모을 때까지 기다립니다.
5. Basic Satellite를 들고 Uplink를 우클릭해 위성을 배치합니다.
6. 빈손 우클릭으로 상태를 보고, Shift + 빈손 우클릭으로 채굴 버퍼를 수거합니다.
7. 중반 이후에는 <ItemLink id="beyondorbit:launch_pad" />와 발사 재료를 써서 더 강한 채굴 미션을 시작합니다.

처음 목표는 간단합니다. **Uplink를 만들고, 위성을 올리고, 쌓인 자원을 가져오는 것**입니다.

## 2. Satellite Uplink 설치

<ItemImage id="beyondorbit:satellite_uplink" scale={3}/>

<ItemLink id="beyondorbit:satellite_uplink" />는 초반 위성 관제 블록입니다.

<RecipesFor id="beyondorbit:satellite_uplink" />

설치 후 서버 틱마다 내부 에너지가 조금씩 충전됩니다. 기본 설정에서는 위성 배치에 에너지가 필요합니다.

## 3. 위성 배치

1. Basic Satellite를 손에 듭니다.
2. Satellite Uplink를 우클릭합니다.
3. 에너지가 충분하면 위성이 배치되고 자동 채굴 미션이 시작됩니다.

## 4. 상태 확인과 수거

- 빈손 우클릭: Uplink 상태 화면을 엽니다.
- Shift + 빈손 우클릭: 위성이 모아둔 자원을 수거합니다.

자원이 바로 나오지 않는다면, 아직 채굴 간격이 지나지 않았거나 대상 천체의 자원이 비어 있을 수 있습니다.
